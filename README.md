# Face Verification with Siamese Network

## Project Overview

This project is a facial verification system built using a Siamese neural network. It detects and compares faces in real time, using a live camera feed to verify whether the detected face matches one of the anchor images in the dataset.

## Author

- Admin

## Key Features

- Real-time live camera face scanning
- Popup window display with face detection boxes
- Face matching against anchor images
- Match result visualization with green/red boxes
- Screenshot capture support during live scanning
- Model training and verification using TensorFlow/Keras

## Main Files

- `face_verification.ipynb` - Main Jupyter notebook containing the full project pipeline.
- `siamese_model.h5` - Saved Siamese model weights.
- `data/` - Dataset directory containing anchor, positive, and negative face images.
- `application_data/` - Additional input and verification image files.
- `training_checkpoints/` - Model checkpoint folder.

## Output and Results

The live scanning output is visible through the popup window generated by the notebook.
During scanning:

- **Green box** indicates a matched face.
- **Red box** indicates a face not matched.
- Real-time status information appears on each frame.

A sample run produced:

- `Total Faces Detected: 23`
- `Matches Found: 22`
- `No Match: 1`
- Live verification results displayed in the popup window and logged in the notebook.

## Usage Instructions

1. Open `face_verification.ipynb` in Jupyter or VS Code.
2. Run the notebook cells in order to load the dataset, build the model, and prepare the verification pipeline.
3. Execute the live face detection cell to open the popup window.
4. Use the webcam view to scan your face.
5. Press `q` in the popup window to stop the scanning.
6. Press `c` to capture screenshots while scanning.
